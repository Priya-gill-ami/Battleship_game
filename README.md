# Battleship_game
This project is a console-based Battleship game built in Python using Object-Oriented Programming. Two players place ships, take turns guessing coordinates, and track hits, misses, and sunk ships. The game follows strict rules and stores results using file handling.

Turn-based gameplay with hit, miss, and ship-sunk detection

Persistent file storage for game results and high scores
